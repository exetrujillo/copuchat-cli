# ¿Qué hay que saber?

Si te metes a la página de latamgpt verás que hay una sección que se llama copuchat, es básicamente un chat que tienes disponible para que ellos recolecten conversaciones y les sirvan para armar el modelo latinoamericano que quieren.

La interfaz de copuchat es bastante mala, por lo que hice esta cli usando su api, que está abierta, generando un UUID random cada vez que ejecutemos los comandos.

Recomiendo usar esta cli en caso de que quieras usar un chatbot desde consola gratis, teniendo en consideracion que los datos se usan. Útil para entornos sin interfaz gráfica y para ahorrar recursos.

Clona este repo y usa los scripts que están en el package.json con npm.
